# Reelhouse Library

Original streaming-style movie library built with Next.js.

Live site:
- https://reelhouse-library.vercel.app

GitHub:
- https://github.com/Franksharpe008/reelhouse-library

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Stack

- Next.js 16
- React 19
- TypeScript
- CSS modules
